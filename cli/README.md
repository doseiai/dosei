# Dosei CLI (Unreleased)

#### Installation

To install the Dosei CLI, run the following command:

Shell (Mac, Linux):

```bash
curl -fsSL https://dosei.ai/install.sh | sh
```

PowerShell (Windows):

```powershell
irm https://dosei.ai/install.ps1 | iex
```
